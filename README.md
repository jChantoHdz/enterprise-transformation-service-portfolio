# Overview

This repository presents a consulting framework structured around **five strategic pillars** and **five engagement tiers**.

The pillars represent the **core transformation domains**, while the tiers represent **depth of engagement**, ranging from executive advisory to full enterprise transformation leadership.

This matrix explains how consulting services evolve depending on the **scope, maturity, and complexity** of the client engagement.

# Descripción General

Este repositorio presenta un marco de consultoría estructurado alrededor de **cinco pilares estratégicos** y **cinco niveles de servicio**.

Los pilares representan los **dominios principales de transformación**, mientras que los niveles representan la **profundidad del acompañamiento**, desde asesoría ejecutiva hasta liderazgo de transformaciones empresariales completas.

Esta matriz explica cómo evolucionan los servicios dependiendo del **alcance, madurez y complejidad** de cada organización.

---
